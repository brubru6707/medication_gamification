Idle Animation Formula:

// Progress builds from 0 to 1 over 2 seconds progress = min(idleTimer - 1.0, 1.0) * 0.5 // Each effect scales with progress squish = sin(time * 0.002) * 0.08 * progress bob = sin(time * 0.001) * 5 * progress sway = sin(time * 0.0008) * 3 * progress rotate = sin(time * 0.001) * 0.02 * progress

Camera Rotation Formula:

// For each platform at depth D: xOffset = sin(cameraAngle) * (D + 1) * 30 yOffset = cos(cameraAngle) * (D + 1) * 10 scale = 1 - abs(sin(cameraAngle)) * 0.1 // Apply to platform position platform.x = originalX + xOffset platform.y = originalY + yOffset platform.scale *= scale
